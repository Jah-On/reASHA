# reASHA
Simple application that automatically reconnect ASHA HA/CIs on boot and done streaming.

## Building

Install `rustup` from your distro's package manager and run `rustup update`. Run `rustup update` even if `rustup` is already installed. Note this does not need to be run from within the project directory. 

```
git clone https://github.com/Jah-On/reASHA
cd reASHA
cargo build # or `cargo run` to run 
```
